# X-Tracker

**X-Tracker** 是一款专为 X、微博 (Weibo)、雪球 (Xueqiu)、小红书 (Xiaohongshu) 和 Substack 用户打造的高效信息聚合与监控 Chrome 扩展程序。

不同于传统的 Timeline，X-Tracker 就像您的私人情报中心，让您可以“只看想看的人，只读想读的内容”。

## 核心功能 (Core Features)

### 双视图设计 (Dual View Design)
- **Feed (动态)**：您的纯净浏览入口。按“最近浏览时间”倒序排列，点击任意账号即可跳转查看。最近点击的账号会自动排在最前，方便您快速回访常用关注。
- **Lists (列表)**：您的管理中心。在这里添加、删除、编辑关注对象，并通过复选框进行批量操作（如一键打开多个选中账号）。

### 多平台支持
- **X & 雪球**：支持完整的内容抓取与聚合。
- **微博 & 小红书 & Substack**：提供“一键直达”功能，点击即可快速跳转至个人主页。
- **普通网页链接**：支持添加任意网页链接（如新闻网站、博客等）。

### 智能管理
- **自定义列表**：创建跨平台的自定义关注列表。
- **别名系统**：为关注对象设置好记的备注名。
- **批量操作**：在 Lists 视图中勾选多个用户，一键批量打开。
- **数据持久化**：所有数据均存储在本地，安全可靠。

## 安装指南 (Installation)

1.  下载源代码并解压。
2.  在 Chrome 地址栏输入 `chrome://extensions/`。
3.  开启右上角的 **开发者模式**。
4.  点击 **加载已解压的扩展程序**，选择解压后的文件夹。

## 使用说明 (Usage)

1.  **添加关注**：在 **Lists** 标签页输入用户名或链接，点击 Add。
2.  **浏览**：切换到 **Feed** 标签页，点击卡片查看内容。列表按最近浏览时间排序。
3.  **批量打开**：在 **Lists** 标签页勾选多个用户，点击底部的 "Open Selected"。
